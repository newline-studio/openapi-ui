# OpenAPI UI

- `UI_URL` \
  public url (no tailing slash)
- `UI_PATH` \
  path to the ui service (no tailing slash)
- `UI_TITLE` \
  title-Tag
- `UI_DESCRIPTION` \
  meta description
- `UI_FILE_PATH` \
  path where the json files are stored
- `UI_SERVICE_{NAME}` \
  list of services with name and filename \
  Services tile and filepath are delimited by a `|` \
  **Example:** `Title|service1.json`
